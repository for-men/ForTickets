# ForTickets

## 프로젝트 소개 
MSA를 기반으로 한 고성능 티켓팅 플랫폼을 개발합니다. 
공연, 콘서트 등의 티켓을 구매할 수 있는 온라인 플랫폼으로, 사용자에게 빠르고 안정적인 티켓팅 서비스를 제공하는 것을 목표로 합니다.

## 팀원 소개

박주창, 이상우, 김용재, 김휘수
